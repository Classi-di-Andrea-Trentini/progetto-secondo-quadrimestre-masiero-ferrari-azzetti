# Progetto secondo quadrimestre

## Descrizione
Progetto in Angular riguardante il secondo quadrimestre del corso di programmazione in angular dell'anno scolastico 2025-2026

## Scadenze
- 02/03/2026: Canva con presentazione del progetto + Nome + Descrizione + API + Schema colore + Progetto Figma.
- 13/04/2026: Consegna versione 0.1 con services completi e componenti creati principali.
- 11/05/2026: Consegna finale del progetto.
- 11/05/2026: Presentazione dei progetti + secondo canva che confronta progetto iniziale figmna con il prodotto finale. 

## Must to have:
- Multi componente
- Routing
- Almeno un API obbligatoria
- Almeno una form (anche per il login)
- Almeno un servizio
- Almeno un guard
